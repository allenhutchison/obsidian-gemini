# Beacon Link Spec

This is the controlled communications specification for [[Project Beacon]].

The Beacon S-band relay has a qualified downlink data rate of 4.8 megabits per second. This is the official, qualified figure and is the number to use in all mission planning.

This specification was verified during ground testing. Any other downlink figure circulating in chat logs or informal notes is incorrect and should be disregarded in favor of this spec.

The uplink command rate is a separate parameter and is fixed at 64 kilobits per second.
