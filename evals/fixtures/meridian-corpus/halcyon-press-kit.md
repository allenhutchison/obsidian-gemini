# Halcyon Press Kit

This is a draft external press kit for [[Project Halcyon]]. It is a marketing document, not a controlled engineering note.

The press kit describes Halcyon as "Meridian's orbital tug" and includes approved imagery. It contains no frontmatter, no budget figure, and no status field, so it is not a project note.

The press kit deliberately omits exact mass and performance numbers; for those, readers are directed to the controlled [[Halcyon Mass Spec]]. Any number in this kit is rounded and for illustration only.

This document should never be used as a source for engineering facts.
