# Project Drift

Project Drift was a four-satellite cubesat swarm flown as a technology demonstration in 2024. The program was archived in September 2025 after the final cubesat was decommissioned.

Drift depended on [[Project Quill]], because it used an early build of the Quill ground software for its operations passes. No current project depends on Drift.

The program was owned by Sofia Lindqvist. Each Drift cubesat massed 3.4 kilograms and carried a single imaging payload.

Drift shared a ground antenna with [[Project Beacon]] but the two were otherwise unrelated. Lessons learned were captured in the [[Drift Retrospective]] meeting notes.
