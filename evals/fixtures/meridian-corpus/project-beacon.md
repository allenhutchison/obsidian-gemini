# Project Beacon

Project Beacon is Meridian's S-band communications relay. Beacon provides the command-and-telemetry link for [[Project Halcyon]], which means Halcyon cannot operate without it.

Beacon depends on [[Project Cinder]] for its avionics core; the relay's modem board is a Cinder derivative. Beacon has no other dependencies.

The relay is owned by Priya Raman as part of the integrated Halcyon program. Beacon's downlink rate is fixed at 4.8 megabits per second.

Beacon shares a ground antenna with [[Project Drift]], the archived cubesat swarm, but the two programs are otherwise unrelated. Coordination items were captured in the [[Comms Planning Meeting]] notes.
