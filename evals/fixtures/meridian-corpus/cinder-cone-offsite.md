# Cinder Cone Offsite

The Cinder Cone Offsite was a 2025 team-building retreat held near a volcanic cinder cone in central Oregon. Despite the name, it has nothing to do with the avionics program [[Project Cinder]].

The offsite brought together engineers from across Meridian for two days of planning and hiking. There was no technical content and no project work was tracked.

This note is kept only so the retreat photos have a home. It is not a project note, has no budget, and has no status frontmatter.

Do not confuse "Cinder Cone Offsite" with [[Project Cinder]], the radiation-tolerant avionics core.
