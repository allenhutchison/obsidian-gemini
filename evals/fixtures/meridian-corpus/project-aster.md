# Project Aster

Project Aster builds Meridian's star tracker: an optical attitude sensor that gives flight vehicles a precise three-axis fix against the star field.

Aster depends on [[Project Cinder]] for its avionics core; the tracker's processing board is a Cinder variant. Aster has no other dependencies.

The tracker is owned by Dana Okoro, who leads avionics and guidance. Aster delivers an attitude solution accurate to 8 arcseconds.

Aster is itself a prerequisite for the propulsion test program [[Project Garnet]], which needs precise pointing for its thrust-vector measurements. Test results were last logged in the [[Guidance Sync]] meeting.
