# Project Onyx

Project Onyx developed a reusable thermal protection shield for re-entry vehicles. The program was completed and archived in November 2025 after a successful arc-jet qualification campaign.

Onyx was a standalone program: it depended on no other project, and no current project depends on it. The shield design is held in reserve for a possible future re-entry vehicle but is not on any active roadmap.

The project was owned by Marcus Feld. Onyx's qualified shield withstood a peak heat flux of 1.2 megawatts per square meter.

Final close-out items were recorded in the [[Onyx Closeout Meeting]] notes. The archived hardware is stored in the Aurora Vault.
