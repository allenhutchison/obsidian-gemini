# Project Quill

Project Quill is Meridian's ground software suite: the mission-planning and telemetry-display tools that operators use from the control room. It is the smallest active-project budget at Meridian.

Quill is a standalone effort. It does not depend on any other project, and only one project depends on it: the archived cubesat program [[Project Drift]] used an early build of Quill for its passes.

The suite is owned by Sofia Lindqvist, who leads mission software and operations. Quill currently supports 12 simultaneous operator consoles.

Quill's planning module was reviewed in the [[Ground Software Review]] meeting. The codebase is written primarily in Rust.
