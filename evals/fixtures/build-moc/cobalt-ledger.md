# Cobalt Ledger

Cobalt Ledger is a project to introduce double-entry accounting to the internal
billing tool so that every charge and refund is traceable. The existing tool
stores only net balances, which makes audits painful.

Scope covers a new ledger schema, a backfill job for historical transactions,
and a reconciliation report. The finance team is the primary stakeholder.
