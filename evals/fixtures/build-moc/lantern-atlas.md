# Lantern Atlas

Lantern Atlas is a project to produce a unified map of every service the
Brindlewood platform runs, including ownership, on-call rotation, and upstream
dependencies. It replaces three stale spreadsheets that nobody trusts.

The first deliverable is an automated dependency crawler; the second is a
searchable web view. Design review is scheduled for next month.
