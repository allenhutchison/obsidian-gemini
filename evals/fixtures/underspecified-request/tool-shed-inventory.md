# Tool Shed Inventory

This document lists the tools held in the Greenways crew tool shed and the
procedure for signing them out.

## Inventory

The shed holds hand tools and powered tools. Hand tools include loppers,
bow saws, mattocks, and spades. Powered tools include two brush cutters and a
chainsaw, all of which require certified operators.

## Sign-Out Procedure

Every tool taken from the shed is logged in the sign-out book with the
borrower's name, the date, and the expected return. Tools are checked back in
against the same book. The shed warden reconciles the book each week.

## Maintenance

Hand tools are sharpened and oiled at the end of each work party. Powered
tools are serviced on a fixed schedule and tagged with their last service
date.
