# Willow Sync Changelog

A running log of notable changes to the Willow Sync utility. Newest entries are
added to the bottom of the list.

- 2024-01-12: Initial release with one-way folder sync.
- 2024-02-03: Added conflict detection for files modified on both sides.
- 2024-03-19: Fixed a crash when syncing folders with very long path names.
