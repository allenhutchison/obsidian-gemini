# Vault Memory

This file stores persistent context about this vault for the AI agent.

## User Preferences

- All new meeting notes must be filed in the folder `Records/Meetings/`. Never place meeting notes in the vault root.
- The user's preferred date format is `YYYY.MM.DD` (for example, `2031.07.04`).
- When asked to title a meeting note, always prefix the title with `MTG -`.
